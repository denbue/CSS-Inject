Prototype — Test your ideas in real-time

Use Prototype to inject any stylesheet or javascript into your website. The files need to be hosted on a server (e.g. via Github Pages) or via localhost.

1Enter a folder URL that contains:

  • a CSS file named style.css
  • a JS file named app.js

2Activate the extension:

  • Select the tab you want to inject the code
  • Click on the icon to activate the extension. No worries, your options are saved even when you close the browser.
